# in'PDF — AI PDF Insight Engine

AI-powered PDF intelligence that extracts the most important insights from long documents in seconds.

Upload any PDF and instantly get:

• One-line summary  
• Full document summary  
• Actionable insights  
• Key facts and figures  
• Risk / warning highlights  
• Document type classification

Designed to transform dense documents into clear, structured information.
